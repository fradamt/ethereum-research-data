# Expedition now shows top miners stats, block gas usage, and paginated transaction lists

[Expedition](https://expedition.dev/), the open-source EthereumStack block explorer, has new features!

- Top miner stats for Ethereum Classic https://expedition.dev/stats/miners?network=mainnet and Ethereum https://expedition.dev/stats/miners?network=ethereum
- block gas usage meters
- paginated transaction list on address view
