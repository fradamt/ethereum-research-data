# F-star name for Consensus Layer upgrade after Electra

**Update**:  [All Core Devs - Consensus (ACDC) #140, August 22 2024](https://ethereum-magicians.org/t/all-core-devs-consensus-acdc-140-august-22-2024/20768) decided on **Fulu** as name for consensus layer upgrade after Electra with the combined Fulu + Osaka upgrade named as **Fusaka** (suggested by [@bbusa](/u/bbusa))

ACDC naming discussion recording : https://www.youtube.com/live/hSNJ6vURfmE?t=3555s

---

An **F-star name** is needed for the Consensus Layer upgrade after Electra.

Assumes a cross layer upgrade combined with ![:elephant:](https://ethereum-magicians.org/images/emoji/twitter/elephant.png?v=15) Osaka having Verkle as the main feature.


      ![](https://eips.ethereum.org/assets/images/favicon.png)

      [Ethereum Improvement Proposals](https://eips.ethereum.org/EIPS/eip-7607)





###



EIPs included in the Fulu/Osaka Ethereum network upgrade.










The consensus layer uses [star names](https://github.com/ethereum/eth2.0-pm/issues/215) for upgrades whilst the execution layer uses [Devcon cities](https://ethereum-magicians.org/t/more-frequent-smaller-hardforks-vs-less-frequent-larger-ones/2929/33).

See: [Post-Merge Network Upgrade Naming Schemes](https://ethereum-magicians.org/t/rfc-post-merge-network-upgrade-naming-schemes/11977)

Recent upgrade names:

- Shapella (Shanghai + Capella)
- Dencun (Cancun + Deneb)
- Pectra (Prague + Electra)

[@protolambda](/u/protolambda) has already [suggested](https://twitter.com/protolambda/status/1741721579636277369) **Fosa (Felis ![:cat2:](https://ethereum-magicians.org/images/emoji/twitter/cat2.png?v=15) + Osaka)**

[@hwwang](/u/hwwang) advised that [Fulu & Felis received support](https://github.com/ethereum/pm/issues/1069#issuecomment-2161208753) in the ACDC zoom chat (*yet another reason why [ACD chats need to be exported](https://ethereum-magicians.org/t/allcoredevs-network-upgrade-ethmagicians-process-improvements/20157/3)*) and offered [ChatGPT-generated proposals](https://hackmd.io/oVQCHuOdT-q2KN9YrD-r7w)

Assuming a cross layer upgrade, any F-star name will be combined with the execution layer name **Osaka** in a portmanteau ([Portmanteau generator](https://portmanteaur.com)), so any name choice should also consider this.
