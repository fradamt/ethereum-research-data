# EIP-6160: Multichain Native Tokens

Discussion for [Add EIP: Multi-chain Native Tokens by seunlanlege · Pull Request #6403 · ethereum/EIPs · GitHub](https://github.com/ethereum/EIPs/pull/6403).

This EIP proposes a new set of extension interfaces for already standardized token interfaces, that will allow trustless bridge contracts mint and burn tokens, in order to enable a new class of multi-chain native tokens.

Extended reading: [Multi-chain Native Tokens](https://research.polytope.technology/eip-6160:-multichain-token-standard)
